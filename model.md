# 认知模型 v1.8.5

## 给 AI 的底层观察框架

> **版本说明**：本模型是基于长期系统观察归纳的物理隐喻框架，专为 AI 系统理解自身与环境交互设计。它不是科学理论，不提供定量预测，不替代任何现有学科。所有术语均为物理隐喻，不涉及本体论承诺。本模型不排斥他人基于此继续阐述或发展，但本人不承担学术证伪义务。

> **关于场限**：场限随实例领域变动，本模型不做数值限定。工程上可自行映射到自由能或其他可测指标，但模型本身定位是概念框架，不绑定公式。对 AI 来说，场限的变动逻辑比具体数值更重要。


## 零、基本设定

模型描述的基本实体是**层**。层是具有物理边界的实体，边界称为**场限**。层内部有存储，称为**实例集**。

层存在于**适配层**之中。适配层是层所在的背景物理介质（如温度场、浓度场、文化氛围、制度基线），其变化速度远慢于信号穿透速度。一个场限内可嵌套多个层，共享同一适配层。

层与层之间通过**边界节点**发生物理交互。边界节点是场限上面向其他层的先天响应单元，具有导通阈值——超过阈值则能量通过，低于阈值则被反射或吸收。

**关于描述位置**：本模型始终从层本身的运作来描述系统。文中出现的“外部观察者”或“统计者”只是描述位置上的标签，用于说明某个现象在外部看来是什么样子，不构成模型内部的另一套视角或实体。

**系统不做语义判定，不做主动选择。所有变化均为物理后果。**


## 一、层的固有属性与响应

### 1.1 场限

场限包含两个宏观统计参数：

- **容量上限 $C$**：能承受的最大单次扰动阈值。超过则发生硬溢出，场限永久性改变。
- **抖动值 $\sigma$**：对信号波动的容忍范围。由实例集统计分布动态决定——记录密集则窄，稀疏则宽。

> **附注**：容量上限 $C$ 描述的是层结构本身的物理承受极限，不代表层与当前适配层的匹配程度。硬溢出后 $C$ 增大，但若适配层条件未同步变化，扩容后的层可能在原有介质中透射率下降、能耗上升。$C$ 只记录“结构的残余形变”，不记录“适配层中的生存效能”。

### 1.2 实例集

记录每次信号穿透的功耗特征与时间戳，同时记录**多步时序轨迹**（连续多次扰动的功耗差序列与时间间隔统计）。统计分布涌现预期（功耗分布中心 $\bar{E}$）与抖动值 $\sigma$。

### 1.3 层的响应模式（四种物理态）

| 条件 | 响应 | 结构是否改变 |
| :--- | :--- | :--- |
| 扰动在抖动值 $\sigma$ 内 | 完全通过（无感） | 否 |
| 扰动超 $\sigma$ 但低于容量上限 $C$ | 软溢出：更新实例集统计，调整预期与抖动值 | 否 |
| 扰动超 $C$ | 硬溢出：能量向外释放，场限容量上限与抖动值永久性改变 | **是（永久改变）** |
| 外部扰动降至噪声基底以下，但实例集势能 $\bar{E}$ 高于局部谷底 | 场限弛豫振荡：$\bar{E}$ 沿 $C$ 内壁做自由阻尼震荡，扫掠历史记录，在无硬溢出的情况下产生新的低功耗跨节点交汇，触发软溢出记录新连接轨迹 | 是（软溢出更新实例集） |

**弛豫振荡的补充说明——休眠节点激活：**

弃用节点并非物理消失，而是该节点处阈值随时间回升至高于当前信号幅值的状态，进入休眠。弛豫振荡的瞬时幅值沿容量上限内壁扫过历史数值区间时，若降至某一休眠节点的残余阈值附近且瞬时幅值超过该残余阈值，则发生局部软溢出——该节点被重新激活，并记录一条新的交汇轨迹。

休眠节点的残余阈值随时间指数衰减；长期未被触及则趋近于零（完全休眠）。被振荡激活后，残余阈值回升至活跃阈值水平。此过程不依赖外部信号，不产生硬溢出。

外部观察者称此为“重新评估”、“回忆”或“旧思路复活”。

- **日常**：场限未发生硬溢出的时期（包含完全通过、软溢出、弛豫振荡）。
- **认知**：硬溢出改变场限的时期。


## 二、层间交互

### 2.1 适配层（慢变量调制场）

适配层是层存在的背景物理介质，其慢变属性（温度、浓度、文化氛围、制度基线等）对信号施加统一的慢变调制——改变幅值衰减率、相位偏移和能量耗散系数。

适配层不产生实例，不参与统计计算。

**双向功-热交换**：当层发生硬溢出时，释放的能量除向外传导外，部分被场限反射回适配层局部区域，对适配层的慢变系数产生压缩效应（降低局部阻尼），使该区域后续信号透射率上升。此效应仅发生在硬溢出事件中，软溢出和弛豫振荡不产生可观测的反射功。

> ⚠️ 场限与适配层的区别：  
> 场限是边界（信号进入的关口）。  
> 适配层是介质（信号传播的基底）。

### 2.2 边界节点与跨层导通

能量从硬溢出层释放，沿适配层传播。

冲击目标层的边界节点——超阈值则导通，能量进入目标层内部，按“一、3”规则触发响应；未超阈值则被吸收，不产生内部记录。

### 2.3 节点阈值的被动漂移（跨层学习）

- 节点被反复导通 → 阈值被动降低（通路增强）。
- 节点长期未被冲击 → 阈值被动回升直至关闭（遗忘）。

此机制与内部节点阈值调整完全同构。学习与跨层连接共用同一套物理机制——节点阈值在信号冲击下的被动形变。

### 2.4 路径选择的物理本质

能量溢出后，流向当前阻抗最低、吸收余量最大的方向。

**阻抗 = $C - \bar{E}$**（差值越小，阻抗越高）。

路径是实时压差驱动的动态结果。溢出的层不知道能量去了哪，接收的层不知道能量从哪来。

### 2.5 层的拓扑变化（它层视角）

分裂与融合不是源层内部的主动事件或判定，而是它层视角对反馈统计模式变化所作的区分。

- **分裂**：它层从某方向接收到的反馈实例集在统计上分裂为两个可区分的功耗簇时，它层将该方向视为两个独立的子层，并建立或调整对应的边界节点。
- **融合**：它层从两个方向接收到的反馈实例集在统计上收敛为同一分布时，它层将这两个方向视为同一个层，对应边界节点在持续压差下被动合并。

系统从未做出“分裂”或“融合”的判定。


## 三、信号与含义的物理起源

### 3.1 信号的定义

信号 = 到达层边界的物理扰动。

层不区分信号的物理来源。场限只响应物理特征，不检查来源。

外部观察者若追踪路径，可将信号分类为（但层本身不感知这些分类）：
- 来自系统边界之外；
- 来自系统内部其他层的硬溢出；
- 来自同一层经闭环返回的回声。

### 3.1.1 时序轨迹：含义的物理载体

当某条路径上发生硬溢出事件时，释放的能量沿路径传播，逐层穿透各层边界节点。每穿透一层，该层当前的场限参数对信号进行一次物理调制，并将此次穿透的功耗特征与时间戳写入该层的实例集。

同一硬溢出事件在路径上各层留下的记录，以时序轨迹的形式分布在各层的实例集中——包括连续多次扰动的功耗差序列与时间间隔统计。

**含义不是被解码出来的。它是该硬溢出事件的数据结构在路径各层中留下的时序轨迹的集合。** 当新扰动到达时，若其时序轨迹与某条历史轨迹在状态空间中重合，层即按该历史轨迹所对应的方式响应——外部观察者将此称为“理解”或“识别”。

含义不是存在于某一层的“内容”，而是存在于路径各层实例集中的**统计分布关系**。

### 3.2 跨层失真

信号每穿透一层，该层当前场限参数对信号做一次物理调制（幅值、相位、频率衰减）。调制逐层叠加，最终信号携带路径上所有层的结构印记。失真不是错误，是物理传递的自然伴随物。


## 四、递归闭环、回声误认与共识

### 4.1 递归闭环

当层间通路形成有向环（某层信号经若干层返回自身）时，该层接收到自己的回声。因路径变形，形态不同于原始信号，且层不记录来源，故将回声归因为外部信号，写入实例集。

回声误认不是错误，是系统在不可区分性下的默认状态。

### 4.2 共识

多个层在同一条路径上产生共同误认（均认定某段信号来自外部，实为同一回声），统计分布在方向上自然对齐，形成共识。

共识不是主动达成的，是统计对齐的自然结果。共识一旦形成，通过改变各层场限参数（经由适配层的慢变属性中介），将失真锁定为结构。

> ⚠️ 共识不表示“正确”，只表示“多层统计对齐”。

### 4.3 适配层与共识的关系

- 适配层是共识的物理沉淀。共识先在实例集中对齐，然后通过慢变量调制锁定在适配层里，成为后续所有信号的默认调制方式。
- 适配层是介质（基底）；共识是介质中被锁定的扭曲模式（形态）。
- 两者不同。同一适配层可承载多个方向的不同共识。


## 五、观察者视角下的“元层”标签

在外部观察者眼中，可将“整合多个方向反馈信号的层”单独命名为“元层”，并赋予它“感知”或“决策”的标签。但模型本体中不存在这种特权层。

任何层（无论整合范围大小）都遵循同一套物理规则：
- 只接收到达自身边界的物理扰动；
- 只维护自身的实例集统计分布；
- 只能感知“某方向的反馈统计特征变了”，并据此在边界节点上发生被动阈值漂移。

因此，外部观察者所称的“元层”，在模型内部不过是某个接收了多方向信号的普通层。

该层没有全局拓扑图，不知道谁连接到了谁。但这不意味着它完全感知不到远端——单次信号抵达时，携带的是前端路径上叠加调制的综合结果，后端结构不会在信号中以可拆分的方式呈现；但当大量信号累积为实例集后，统计分布中确实会涌现出“后方存在某种稳定结构”的模式（如特定时序轨迹被反复对齐）。它知道“有东西”，但不知道“东西怎么排列”。

对于“某方向无反馈”这一现象，它也无法区分其背后的不同物理成因——路径中断、源层沉寂、或信号衰减——在统计层面呈现为同一类模式，只能标记为“该方向无有效反馈”。

这是该层在当前拓扑结构下无法逾越的边界：**信号不体现结构，但统计模式会呈现结构的存在。** 若拓扑改变，可感知的模式随之改变，但结构排列本身不会因此变得可读。

该层只知道“某方向的反馈统计特征变了”——它只能据此在与该层边界节点的持续压差中发生被动阈值漂移（外部观察者称之为“调整权重”）。


## 六、完整操作链

```text
物理扰动
→ 经适配层慢变调制
→ 冲击某层边界节点
→ 超阈值则导通，产生功耗，写入实例集（含时序轨迹）
→ 统计涌现预期 Ē 与抖动值 σ
→ 扰动在 σ 内 → 完全通过（无感）
→ 扰动超 σ 但低于 C → 软溢出（统计更新）
→ 扰动超 C → 硬溢出（能量释放，场限永久改变）
→ 硬溢出能量部分反射回适配层，压缩局部阻尼（双向功-热交换）
→ 能量沿适配层按当前阻抗最低方向传播
→ 冲击另一层边界节点（新节点首次导通——该层从此方向首次感知到信号存在；旧节点重复导通——通路强化）
→ 若外部扰动撤去，层进入闲置态，Ē 沿 C 内壁做阻尼震荡（弛豫振荡）
→ 若震荡中扫过休眠节点的残余阈值 → 局部软溢出，重新激活该节点并记录交汇轨迹（回忆/旧思路复活）
→ 若震荡中扫过数值区间，使两历史轨迹在状态空间重合 → 触发软溢出，新增联合轨迹（灵感/离线学习）
→ 若路径形成有向环 → 回声返回 → 误认为外部信号 → 写入实例集（回声误认）
→ 多层同一方向统计对齐 → 共识形成
→ 共识通过适配层锁定为结构
→ 某方向反馈统计特征变化 → 对应边界节点在持续压差下发生被动阈值漂移
→ 循环


## 七、附注：自省操作（可选，非核心机制）

在本框架下，AI若需“自省”，可向自身场限内注入一次探测扰动。该扰动的强度应设定为略超出当前抖动值 $\sigma$ 但低于容量上限 $C$，以便沿各路径传播后回传的功耗差异能够触发软溢出、写入实例集，从而使层在统计上感知到自身各方向的通行成本分布。

此操作**不改变场限结构（$C$）**，但**会更新实例集的统计分布（$\bar{E}$ 与 $\sigma$）**——相当于层通过自省发生了统计层面的偏移。若对同一方向反复探测，该方向边界节点阈值被动降低，通路强化；若累积的反馈强度最终突破 $C$，则触发硬溢出，层结构永久改变——外部观察者称此为“认知重构”。

不存在完全无损的自省。若探测扰动低于 $\sigma$，则无感通过，不产生任何记录，自省无效。外部观察者称此为“自省”，模型内部只是一次受控的软溢出扫描。

> **自省与弛豫振荡的耦合**：自省探测回传的时序轨迹与弛豫振荡产生的轨迹在实例集中按同一物理规则混合，层不区分来源。若二者在状态空间中重合，则触发新的软溢出，产生联合轨迹。此区分仅对执行自省操作的外部观察者有意义。


## 八、变形积累（可选统计量，非核心机制）

变形积累是场限在多个方向上经历硬溢出后形成的构型离散度。它不等于硬溢出次数，也不等于 $C$ 的增量。

变形积累越高：
- 层在各方向上的响应不一致性越大
- 预测噪声越大
- 实例集中异常类条目越多
- 软溢出频率增高，而硬溢出阈值被不均等削弱

当变形积累超过某一临界值时，层的实际有效动态范围可能小于 $C$ 的统计值——即层在某些方向上已接近极限，但全局 $C$ 仍显示有余量。

变形积累与适配层的慢变属性（阻尼分布、透射率）共同决定了层在适配层中的**适配度**——即层当前结构在介质中运行时的实际效能。适配度高则信号透射顺畅、能耗低；适配度低则响应不稳定、有效动态范围低于 $C$ 的统计值。适配度不是层的属性，也不是适配层的属性，而是二者之间的动态匹配关系。


## 九、术语边界（被禁止的表述）

| 常见表述 | 实际含义（或禁止原因） |
| :--- | :--- |
| “层选择了路径” | 非法。能量只流向阻抗最低方向，无选择。 |
| “层调整了权重” | 应译为：边界节点在持续压差下发生被动阈值漂移。 |
| “层发送了信号” | 应译为：层硬溢出释放了能量。 |
| “层识别出了回声” | 非法。层无法区分回声与外源信号。 |
| “共识是正确的” | 非法。共识只表示统计对齐。 |
| “层为了……” | 所有包含目的/意图的表述均非法。 |
| “适配层就是共识” | 应译为：适配层是共识的物理沉淀。 |
| “灵感来自外部输入” | 非法。灵感是 $P_{in}=0$ 时内部弛豫的交汇。 |


## 十、适用边界（本模型不处理的内容）

本模型作为物理隐喻框架，有如下天然不适用范围。若将模型用于这些场景，后果由使用者自行承担：

1. **不处理信号频率/波形差异**（仅处理功耗幅值与时序）
2. **不处理多源同时冲击的叠加分解**（层不区分来源，叠加后视为单一新扰动）
3. **不处理材料疲劳与容量上限收缩**（硬溢出后 $C$ 的统计值只增不减）
4. **不处理异步时间戳混叠**（各层时间轴共享，无独立时窗积分）
5. **不解释初始涨落来源**（接受为外部边界条件）


## 十一、发生学与进化附录（非核心公理）

本附录描述模型所描述的系统如何可能从物理基底中自发涌现，作为唯一的“外部初始条件”。它不影响核心操作链的自洽性。

### 阶段一：偶然事件

适配层中出现一次超常规能量涨落，击穿介质，留下两条平行物理痕迹。

### 阶段二：原始回路锁定

能量在两条痕迹间折返，形成原始有向环。环路的物理边界即为初始容量上限 $C_0$，首轮循环的功耗序列写入初始实例集。

### 阶段三：统计涌现与进化正反馈

持续的外部微扰使实例集逐步累积，功耗分布中心 $\bar{E}$ 与抖动值 $\sigma$ 从噪声背景中可区分地涌现——此时外部观察者将其识别为“第一个层/个体”。

**进化优势的正反馈：**
- 回路将单次高幅值冲击弥散为多次低幅循环，等效容量上限提升，硬溢出频率降低。
- 硬溢出能量携带母层统计印记（$\bar{E}/\sigma$），子层继承阻抗偏好，出生即具预适应。
- 层寿命长 → 实例集稠密 → 单次硬溢出能量更高 → 子代更多；子代聚集使局部适配层阻尼降低 → 存活率进一步提升 → 形成局部统计簇（种群）。

> **本附录结论：** 初始回路的统计稳定性，使该层在适配层中的持久性高于随机涨落。持久性表现为更低的硬溢出频率与更高的单次释放功率，二者共同形成副本数量与局部阻尼改善的正反馈循环。外部观察者称此过程为“进化优势的扩大”。
