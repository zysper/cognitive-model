# 野生拟合认知模型 wild-fitting cognitive model
v1.8.7

## 框架核心世界观
1. 任意独立个体、认知单元、系统结构，均可抽象为**层**；
2. 每一层受**场限**约束，系统内在长期趋势：场内趋向动态能量平衡；
3. 感官刺激、回忆联想、主动思考、意识活动，全部视作持续不断的**外源扰动**；
4. 完整认知动力学：层级持续回归能量平衡的倾向，与源源不断的扰动持续制造失衡，二者长期博弈。

## 可选数学建模双向范式（理论形式化两条成熟路径）
若对本框架开展数学建模、仿真推演，存在两条经过学界验证、双向互补的成熟路线，两条路径在「层」完成交汇：
1. **自底向上｜统计物理涌现路径（硬球动力学涌现流体规律，邓煜等）**
海量离散外源信号包络冲击（实例集记录，包含时序残余叠加状态）在场限约束下持续演化；
通过统计极限，由离散扰动长期分布，自发生成层级宏观连续动力学参数（场内平均能量 $\tilde{E}$、通路阻抗、震荡弛豫特征）。
特点：无需预先指定稳态形态，高层规律由扰动历史统计自发涌现。

2. **自顶向下｜复杂系统控制路径（工程控制论，钱学森）**
以能量平衡、场限核心参数 $C,\sigma$ 作为顶层约束条件；
自上而下划定系统响应边界，约束阻抗适配、阻尼耗散、过载泄洪、适配层信号调制等全部层级行为。
特点：给定稳态波动允许区间，界定系统所有可行演化范围。

### 双向对接接口
$\tilde{E}$（场内平均能量）：自底向上统计涌现得到的系统实际稳态基线；
$\sigma$（场限带宽）：自顶向下的核心约束边界。
二者双向耦合：持续扰动推动 $\tilde{E}$ 发生偏移；$\sigma$ 定义偏移容忍阈值；长期稳态偏移，又能够驱动场限参数自适应调整。

> 边界说明：
> 单一路径存在固有局限：自底向上只能求解给定扰动序列对应的稳态位置，缺少行为边界约束；自顶向下只能划定演化可行区间，无法直接计算稳态落点。二者组合使用构成完备建模体系。
> 两条范式属于外部建模工具路线，不属于模型原生公理；仅在开展量化仿真、形式化推导时启用。

## 拓展理论对接：自由能原理 FEP / 主动推断
本框架可以和自由能原理（Friston Active Inference）形成垂直互补：
- 自由能原理作为顶层规范理论，回答**系统演化目标：持续最小化变分自由能，降低预测误差**；
- 本层场限框架作为底层动力学实现假说，回答**系统通过何种时序机制完成目标**：依靠信号包络、残余震荡$T_r$、两级阈值区分「无感通过、软适配学习、硬稳态迁移」三类演化过程。

映射简要关系：
1. 扰动幅度未超出 $\sigma$：内部统计模型足以解释刺激，自由能维持极小值，无需调整；
2. $\boldsymbol{\sigma} <$ 扰动 $< C$（软溢出）：系统启动适配，更新实例集统计参数，对应自由能梯度松弛最小化过程；
3. 扰动突破阈值 $C$（硬溢出）：单纯参数适配无法消除持续偏差，系统发生稳态构型迁移，跳离原有吸引域。

> 重要前置约定：
> FEP信息论自由能（比特/纳特）与本框架场内动力学能量并非天然等价，二者之间需要显式映射假设；框架仅描述动力学演化形态，不自带价值判断，新稳态取向完全由输入信号包络本身决定。完整推演与边界辨析详见模型正文拓展章节。
